Que yo sepa cada sáb GPS tiene un reloj atómico muy preciso, y cada uno en la constelación se sincroniza con el llamado "tiempo de GPS". Una vez que el receptor tiene 4 o más satélites, que resuelve (por algún método) una ecuación donde hay cuatro incógnitas: x, y, z, b; donde (x, y, z) es la ubicación del receptor, y b es la diferencia de hora entre el reloj interno del receptor y el tiempo (correcto) GPS. NTP señal no se utiliza directamente para obtener la posición GPS. Pero para una solución exacta que usted necesita relojes muy precisos. Estamos hablando aquí nanosegundos. Los satélites GPS se transmiten actual tiempo GPS, pero incluso a medida que viaja a la velocidad de la luz, hay una cierta demora. GPS receptor no tiene manera de saber lo que el retraso es, por lo que tiene a la aproximación de varias señales recibidas. Con cada transmisión de información recibida el reloj de obtener mayor precisión. Así que el mejor momento que tiene en el principio, la menos señales de tiempo que usted tiene que recibir para tener un reloj de precisión.

¿Quién dirige estos servidores NTP?

Los servidores NTP son administrados por el pool.ntp.org proyecto </ a>, que es un grupo virtual muy grande de servidores de horario.

¿Por qué la lista contienen regiones como Yugoslavia?

La lista no representa las actuales regiones políticas, porque esto pede romper configuraciones existentes utilizando servidores NTP del proyecto pool.ntp.org mira la lista como un conjunto de localizaciones geográficas.
